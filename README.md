This project is a full-stack YouTube clone built using the MERN stack as part of the Internshala Full Stack Development Capstone Project.

##  Tech Stack
- Frontend: React.js, React Router, CSS
- Backend: Node.js, Express.js
- Database: MongoDB Atlas
- Authentication: JWT
- API Client: Axios

##  Features
- User authentication (Register / Login)
- Create and manage channels
- Upload videos with thumbnail and category
- Watch videos with:
  - Like / Dislike functionality
  - Subscribe / Unsubscribe channel
  - Live subscriber count
  - Show more / Show less description
  - Comment system
  - Recommended videos sidebar
- Fully responsive YouTube-like UI

##  Pages
- Home Page (video feed)
- Video Watch Page
- Channel Page
- Login & Register Pages
